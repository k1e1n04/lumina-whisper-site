Installation

1. Purchase

Purchase a license from Polar.sh. After purchasing, you will receive a license key by email.

2. Download

Download the latest LuminaWhisper.dmg from GitHub Releases.

3. Installation

  1. Open the downloaded .dmg file.
  2. Drag the LuminaWhisper icon to the Applications folder.
  3. Launch LuminaWhisper from the Applications folder.

4. License Activation

An onboarding wizard will appear on first launch. The first step will prompt you to enter your license key.

Enter the license key you received after purchasing on Polar.sh and click Activate.

5. Input Monitoring Permission

Input monitoring permission is required to detect fn key presses. Click Allow in the wizard.

6. Microphone Permission

Microphone access is required to record audio. Click Allow in the wizard.

To change the permission later, go to System Settings → Privacy & Security → Microphone and enable LuminaWhisper.

7. Accessibility Permission

Accessibility permission is required to insert transcription results directly into the active app. Click Allow in the wizard.

8. Speech Recognition Model Download

The speech recognition model (approx. 150 MB) will download automatically. A Wi-Fi connection is recommended. The wizard will proceed automatically when complete.

9. Select Transcription Language

Select the language you will speak. You can change this at any time in Settings.

Requirements: macOS 14.0 (Sonoma) or later · Apple Silicon (M1 or later)
