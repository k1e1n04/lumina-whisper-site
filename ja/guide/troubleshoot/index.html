Troubleshooting

Microphone Not Detected

  1. Open System Settings → Privacy & Security → Microphone
  2. Check that the LuminaWhisper toggle is on
  3. Restart LuminaWhisper

Hotkey Not Working

  1. Open System Settings → Privacy & Security → Accessibility
  2. Add LuminaWhisper to the allowed list (use the + button if not present)
  3. Restart your Mac

App Won't Launch

  1. Verify you are running macOS 14 or later on Apple Silicon
  2. Check that the app is allowed to run under System Settings → Privacy & Security
  3. Delete the app and reinstall

Text Not Being Inserted

  1. Make sure a text input field has focus before stopping recording
  2. Check that accessibility permission has been granted
  3. Enable 'Clipboard Fallback' in Settings — if insertion fails, you can still paste manually with ⌘V

If the issue persists, try uninstalling and reinstalling the app, or reset your settings.
