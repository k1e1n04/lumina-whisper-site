Mac Voice Recognition

声をテキストに、瞬時に。

Lumina Whisper は、Mac で高精度な音声入力を実現するアプリです。OpenAI Whisper をベースに、ローカル環境でも高速かつ高精度な文字起こしが可能です。

How it works

Claude Code にも、声で指示できる

どのアプリでも、カーソルがある場所に直接テキストが挿入されます。クリップボードを経由せず、キー一発で完結。

  1. 1入力したい場所にカーソルを置く
  2. 2Fn を押して話す
  3. 3再度 Fn → テキストが挿入される

Features

なぜ LuminaWhisper？

01

ホットキー一発

Fn キーでいつでも即座に音声入力を開始。アクティブなウィンドウへ直接テキストを挿入します。

02

完全ローカル処理

音声データはデバイス外に出ません。インターネット接続不要でプライバシーを守ります。

03

Apple Silicon 最適化

Neural Engine をフル活用した高速・省電力な音声認識。

04

LLM 後処理

「えっと」「あの」などのフィラーを自動除去し、カスタム指示で整形します。完全ローカルで動作します。

05

13言語対応

日本語・英語をはじめ、中国語・韓国語・フランス語・ドイツ語・スペイン語・ポルトガル語・ロシア語・アラビア語・ヒンディー語・イタリア語など13言語に対応。
