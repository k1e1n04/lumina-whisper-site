リリースノート

v0.2.0

2026-04-04

新機能

  • 初回起動時のセットアップウィザード — アクセシビリティ・マイク権限の付与とホットキー設定をガイドに従って完了できます
  • モデルウォームアップ画面 — 初回録音前のモデル準備中に進捗を表示し、準備完了まで録音を安全にブロックします
  • オンボーディングにアクセシビリティ権限・言語選択ステップを追加 — セットアップウィザード内で権限付与と認識言語の選択が完結します
  • デフォルトホットキーを Fn に統一 — 新規インストール時のデフォルト設定が Fn になります
  • 認識言語を13言語に拡大 — 日本語・英語に加えて、中国語・韓国語・フランス語・ドイツ語・スペイン語・ポルトガル語・ロシア語・アラビア語・ヒンディー語・イタリア語をサポート

バグ修正

  • マイク権限を起動時に事前取得するよう変更 — 初回録音時に権限ダイアログが表示されない問題を修正
  • RMS による無音検出を導入 — 無音時にハルシネーションが挿入される問題を修正
  • テキスト挿入のペースト遅延を増加（50ms → 200ms）— 挿入が抜ける問題を修正
  • モデルダウンロード後に設定を保存するよう変更 — 初回起動時に文字起こしエンジンが初期化されない問題を修正
  • 言語選択オプションのタップ判定を修正 — contentShape を追加してタッチ領域を正しく設定

v0.1.0

2026-04-02

新機能

  • Fn ホットキーによるワンキー音声入力 — アクティブなウィンドウへテキストを直接挿入
  • WhisperKit 搭載の音声認識エンジン — Apple Neural Engine をフル活用した高速・高精度な文字起こし
  • テキスト直接挿入とクリップボードフォールバック — 挿入に失敗した場合でも ⌘V で手動貼り付け可能
  • モデル選択（tiny / base / small / medium）と言語設定（auto / ja / en）
  • ローカル LLM による後処理 — フィラー自動除去・カスタム指示・アプリ別指示に対応
  • テキスト置換ルール — 文字起こし後のテキストに対して任意の置換を自動適用
  • 単語辞書 — 固有名詞を登録して Whisper の認識精度を向上
  • 文字起こし履歴 — 過去の入力結果を一覧表示・再利用
