Basic Usage

Voice Input Steps

  1. 1

    Focus a text input field

    Click where you want to type (a text editor, email body, etc.).

  2. 2

    Press Fn

    Recording starts. The menu bar icon blinks to indicate recording.

  3. 3

    Speak

    Speak clearly. Punctuation is inserted automatically.

  4. 4

    Press Fn again

    Recording stops. The recognized text is automatically inserted at the cursor position.

Tip: Accuracy is highest in quiet environments. Using an external microphone is also recommended.
